🎯 Project Features

RWKV Agent Kit is an intelligent agent toolkit based on the RWKV model, focused on providing AI agents with true memory and reasoning capabilities.

Core Advantages

  • True Memory System: Not just context windows, but long-term memory based on vector databases
  • Intelligent Agent Architecture: Complete agent framework supporting task planning, execution, and reflection
  • Highly Extensible: Modular design for easy addition of new features and tools
  • Production Ready: Comprehensive error handling, logging, and performance monitoring

Quick Experience

# Add dependency
cargo add rwkv-agent-kit

# Or add to Cargo.toml
[dependencies]
rwkv-agent-kit = "0.1.1"
use rwkv_agent_kit::RwkvAgentKit;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let agent = RwkvAgentKit::new("config.toml").await?;
    
    let response = agent.chat("Hello, please introduce yourself").await?;
    println!("AI: {}", response);
    
    Ok(())
}
